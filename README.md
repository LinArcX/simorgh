# Simorgh
Another line editor.

## Tools
- git
- gcc(or clang)
- gdb(or seergdb)
- fzf

# Build/Run/Clean
## Linux
First source **project.sh** file:
`. project.sh`

And then call `p`(it's a bash function).

Now you can see a list of commands, like:
- build
- run
- clean
- debug
- search
- search/replace
- generate tags
- valgrind
- find strings in the binary
- list symbols from object files
